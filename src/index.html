Excusa Generada:

Excusa de ejemplo muy pero que muy larga

Un programa que genera excusas aleatorias es una web divertida y útil que crea explicaciones o justificaciones improvisadas para diversas situaciones. El programa puede estar diseñado para producir excusas humorísticas, absurdas o plausibles para cosas como llegar tarde al trabajo, olvidar una tarea o evitar compromisos sociales.
